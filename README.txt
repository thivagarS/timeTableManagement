Time Table management
Steps to run the program 
1 --> npm install (to download the packages used for the front end.)
2 --> Output for Question 1 : node index.js 1 6 (where 1 -- stands for question number and 6 -- stands for class for which we need to generate time table 6 to 10)
3 --> Output for Question 2 : node index.js 2 (where 2 -- stands for question 2)
4 --> Output for Question 3 : node index.js 3 

External packages are used for only displaying the data in the terminal
